# Frontend (pendiente)

El frontend (simulando un chat de WhatsApp para probar los modelos en vivo)
todavía no está implementado — se construirá con un framework moderno,
consumiendo la API en `code/back/` (endpoints `/analizar`, `/intencion`,
`/sentimiento`, `/modelos`).

Mientras tanto, la API se puede probar directamente en `http://localhost:8000/docs`
(Swagger UI autogenerado por FastAPI) una vez levantado el backend.
